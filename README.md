# Doubly Linked List in C
This repository contains a **C program implementing a doubly linked list**. The program demonstrates:
- Inserting nodes at the **head** of the list
- Inserting nodes at the **tail** of the list
- Inserting nodes **after a specific node** (middle insertion)
- Traversing the list **forward and backward**
## Features
- Doubly linked list with `prev` and `next` pointers
- Forward and backward traversal
- Easy to modify and extend for deletion or search functions

## How to Run
1. Open terminal or your favorite C IDE (like Code::Blocks, Dev-C++, or VS Code).  
2. Compile the program:
gcc doubly_linked_list.c -o doubly_linked_list
Run the executable:
./doubly_linked_list   # On Linux/Mac
doubly_linked_list.exe # On Windows
**You will see the list printed forward and backward. ---> Example Output:**
                         **Forward: 30 -> 20 -> 99 -> 10 -> 40 -> 50 -> NULL**
                         **Backward: 50 -> 40 -> 10 -> 99 -> 20 -> 30 -> NULL***
